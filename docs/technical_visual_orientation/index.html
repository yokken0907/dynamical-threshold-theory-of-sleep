1. DTTS-specific sleep mission variable

In this repository, the mission variable is not sleep-quality scoring, medical diagnosis, insomnia treatment, or clinical prediction. It is sleep-like recovery-transition diagnosis: a reduced dynamical system crosses from wake-like persistence toward a sleep-like recovery basin when homeostatic load, arousal/activation, adaptive threshold, synchronization, and noise move into a transition-compatible configuration.

2. Reduced dynamical-threshold model

DTTS should be read as a reduced conceptual/theoretical dynamical model of threshold-mediated sleep-like transition. It does not define a clinical biomarker, diagnosis rule, treatment protocol, wearable-device algorithm, or patient-specific predictor. The useful object is the diagnostic map of load, arousal, threshold adaptation, synchronization, noise, and recovery-state stability.

DTTS-specific model reading

  • The state channels are conceptual dynamics: load, arousal, adaptive threshold, synchronization, noise, and recovery basin.
  • The model is reduced and theoretical; it is not a validated clinical model.
  • The value is in transition interpretation and claim-bounded dynamical analogy.
  • Diagnosis, treatment, patient prediction, and clinical validation are outside scope.

3. Transition channels: load, arousal, synchronization

The differentiating feature of DTTS is that failure or success has threshold-dynamics meanings: homeostatic load can approach the threshold, arousal can keep the system wake-like, synchronization can stabilize transition, noise can fragment recovery, and adaptive threshold drift can change when the transition occurs.

4. Regime split: wake-like persistence versus recovery transition

A project-specific strength of this archive is the distinction between dynamical regimes: wake-like persistence, near-threshold accumulation, threshold crossing, sleep-like recovery consolidation, and fragmented/relapse-like return. These are conceptual diagnostic categories, not medical classifications.

Diagnostic interpretation

  • Wake-like persistence is a model state, not a diagnosis.
  • Threshold crossing is a conceptual transition, not a clinical sleep onset marker.
  • Fragmentation risk is not a clinical disorder classification.
  • Clinical validation and medical use remain separate requirements.
