# Dynamical Threshold Theory of Sleep — Technical Visual Orientation

This folder contains a browser-only technical visual orientation page for the `dynamical-threshold-theory-of-sleep` repository.

Open:

```text
docs/technical_visual_orientation/index.html
```

## Purpose

This page gives technically interested first-time readers a compact, DTTS-specific visual map of the repository before they read the manuscript, figures, source materials, and medical claim-boundary documents.

It summarizes:

- the DTTS-specific mission variable,
- sleep-like recovery-transition diagnosis,
- homeostatic load, arousal/activation, adaptive threshold, synchronization, and noise channels,
- wake-like persistence, threshold crossing, recovery basin, and fragmentation risk,
- medical non-claim discipline,
- evidence hierarchy,
- repository reading order,
- explicit diagnosis / treatment / clinical-validation non-claims.

All graph sections include replay controls so the reader can step through the visual logic interactively.

## Non-claims

This page does not claim:

- medical diagnosis or clinical screening,
- sleep-disorder treatment recommendation,
- patient-specific prediction,
- clinical validation,
- wearable-device algorithm certification,
- medical-device or therapeutic readiness,
- replacement for medical or sleep-specialist evaluation.

It is an orientation aid only and must not be cited as clinical, diagnostic, therapeutic, medical-device, or patient-care evidence.
