# Medical claim boundary

DTTSは睡眠を扱うが、医療モデルではない。

## 禁止表現

- 睡眠障害を診断できる
- 不眠症を改善できる
- 睡眠治療に使える
- 個人の睡眠状態を予測できる
- 臨床的に検証済み

## 推奨表現

- mathematical network prototype
- recovery phase transition
- adaptive oscillator network
- proof-of-concept simulation
- qualitative theoretical model

## 公開上の一文

DTTSは、睡眠様遷移を適応的ネットワークの回復相ダイナミクスとして記述する数理プロトタイプであり、臨床睡眠モデル、医療ツール、ヒトデータ検証、または完全な生物学的睡眠理論を主張するものではない。
