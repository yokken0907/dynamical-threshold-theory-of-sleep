# 再現性メモ

## 現在の状態

このリポジトリはpaper companion archiveである。

## あるもの

- 論文PDF
- selected simulation scripts
- selected figures
- CSV outputs
- prediction summary
- claim boundary / limitations

## あえて入れていないもの

- virtual environment
- compiled dependencies
- site-packages
- pycache

## 再現性の意味

DTTSの再現性とは、同じ抽象network model、drive、threshold dynamics、coupling update、protection fieldの下で、同等のqualitative simulation patternsを再生成できることを意味する。

実脳データや臨床データの一致を意味しない。
