## Scope

The Dynamical Threshold Theory of Sleep (DTTS) is a mathematical and computational prototype theory. It represents sleep-like transitions as recovery phase dynamics in adaptive multicellular networks.

The framework combines:

- adaptive phase oscillators,
- local homeostatic load,
- local sleep propensity,
- noisy wake-like perturbation,
- selective protection of coupling structure,
- and bounded sleep-dominant free-energy relaxation.
